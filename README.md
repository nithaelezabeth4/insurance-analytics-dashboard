##  Project Overview
This project is a Tableau dashboard analyzing insurance data to find insights on policies, insured value, and risk factors.

##  Tools Used
- Tableau
- Excel

##  Key Insights
- Average insured value analysis
- Policy count distribution
- Risk factor comparison (Flood, earthquake)

##  Files
- insurance analytics pjt.twbx (Tableau dashboard)
- data insurance.xlsx (dataset)

##  Outcome
Built an interactive dashboard to support data-driven decision making in insurance domain.
